# About
Netherite Ascension will add a new tier of armour and tools called Divine Netherite, Finally allowing the user to instamine deepslate, a much requested feature from the Minecraft community.
Development is currently in progress and release is estimated at Q3 2026.
## Setup

Mod Built for Minecraft Fabric 26.1

Fabric API is required.
